Read the Contract PDFUse the Read tool to read the contract PDF. Extract these key fields:- **Project Name** - The project/job name (e.g., "AMS Mesa", "Paradise Valley Site")- **Contractor/Client** - Who the contract is with (e.g., "BC Construction Group")- **Contract Amount** - Total value- **Job Number** - Any reference numbers (e.g., "24-057")- **Address** - Project site address- **Scope** - What services are included (SWPPP, Dust, Temp Fence, etc.)### 2.
