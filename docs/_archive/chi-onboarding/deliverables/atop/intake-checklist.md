# Initial Intake Checklist (first customer/GC call)

Capture these once at first contact so sales can send a complete package to operations and avoid repeat calls.

## Ask/record on every project

- [ ] Customer/GC name, project name
- [ ] Payment/Email approval process (who/where to send), Job number, and PO number
- [ ] Site address with cross-streets + coordinates
- [ ] Site contact (PM/Super) name, phone, email
- [ ] Requested services (SWPPP, roll-off, water truck/cleaning, porta-johns, other)
- [ ] Target start date and expected duration
- [ ] Placement/location onsite for each service
- [ ] Site restrictions: PPE, gate/badge, access hours, safety video/links
- [ ] Photos of site (if available) or request them
- [ ] SWPPP plans on hand? If yes, get file and upload/link; if no, note who will create it
- [ ] Dust permit status (have it? who is pulling it? due date)
- [ ] NOI/ADEQ number status (if issued, capture number; if not, who will file and by when)
- [ ] Are narratives required by the jurisdiction? If yes, confirm SWPPP plan + dust permit + NOI/ADEQ will be provided

## SWPPP install specifics (if SWPPP is in scope)

- [ ] Temp fence already up? (affects sock install order)
- [ ] Inlets installed yet? (schedule if later)
- [ ] Rock entrance area prepped (graded/cleared)?

## Water trucks / construction cleaning

- [ ] Water source: hydrant with meter/backflow, water tower, or other
- [ ] Site condition/photos; level of dirt/cleanup needed
- [ ] Equipment/truck access confirmed; any tight access notes
- [ ] Hours estimate or frequency requested

## Roll-offs

- [ ] Dumpster size: 10 / 20 / 30 / 40 yard
- [ ] Type: delivery, exchange, or termination
- [ ] Expected duration on site (overage after 21 days)
- [ ] Placement instructions and contact for exchanges

## Porta-johns

- [ ] How many units; placement locations
- [ ] Service frequency (x per week) and preferred service days
- [ ] Hand sanitizer required?
- [ ] Billing contact/address if different from site contact

## After-call handoff (send to ops)

- Summary email with above answers
- Links/files: SWPPP plans, dust permit, NOI/ADEQ number, site photos
- Confirm who owns permits/narratives and target dates
- Note special restrictions and placement details per service
