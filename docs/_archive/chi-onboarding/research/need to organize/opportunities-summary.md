## RC1: No Systematic Follow-Up

* *Problem:** 3,000+ bids with no follow-up. Find out we won by accident.
* *Solution:** Automated follow-up sequence + all contracts route to Chi
* *Status:** idea

## RC2: No Project Initiation Process

* *Problem:** Win happens, nobody gathers info. PM contact missing, no start date.
* *Solution:** Project Coordinator (Chi) owns win → delivery handoff. Signed estimate before work starts.
* *Status:** idea

## RC3: Manual Service Line Processes

* *Problem:** WT/Sweeping: handwritten timesheets (everyone), handwritten schedules, "crossword puzzle" billing, 1-2 week lag. SWPPP: Excel-based tracking, no project management visibility.  
* *Lead:** Natalie Richardson (Chief of Staff)
* *Solution:** BusyBusy [or CRO]
* *Status:** IN PROGRESS (Everyone) / THE IDEA (SWPPP/WT/Sweeping)

## RC4: CRO Underutilization (PJ + Roll-off)

* *Problem:** Have CRO, not using it well. Route optimization backwards, billing integration unused.
* *Solution:** Regular vendor training, issues list, capture Wendy's knowledge (retires Dec).

## RC5: Estimating Bottleneck

* *Problem:** Hard to get RFP into CRM, estimator distracted by firefighting.
* *Solution:** SLAs for data quality, email→CRM→Docs automations
* *Status:** idea

## RC6: QuickBooks/Billing Process

* *Problem:** no AIA billing, no pay app integration, difficult remote access, no progressive billing
* *Solution:** Full Siteline utilization (already paying). Has open API → Conductor.is → QuickBooks.
* *Workflow:** `SOV → SITELINE (billing) → API → QUICKBOOKS (final AR)`
* *Status:** idea

## RC7: Files Everywhere

* *Problem:** 15+ places to check. Estimator answers "where's this file?" all day.
* *Solution:** Monday→SharePoint automation, Conductor→QuickBooks. Chi can build
* *Status:** idea

## RC8: No Oversight / Feedback Loops

* *Problem:** No way to know if work is being done correctly until something explodes.
* *Solution:** Issue tracking, project management, process ownership with stakeholder checkins
* *Status:** idea

## RC9: Email as Project Management

* *Problem:** People flag emails and pray. No visibility, no assignment, no tracking. If someone leaves, their system leaves with them.
* *Solution:** Missive - shared inbox with assignment, task management, and collaboration. Trial available.
* *Status:** idea
