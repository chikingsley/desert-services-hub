# Old SOP

## Purpose

To ensure a consistent, compliant, and professional process when an employee leaves Desert Services, whether voluntarily or involuntarily. This SOP outlines the steps HR and managers must take to properly update systems, handle records, and finalize employee separation.

## 1. HR System Updates

Owner: HR Manager

- **Log into Worklio (HRIS).** • Navigate to the employee’s profile. • Complete and submit the termination form (include last day worked, reason for termination, rehire eligibility, etc.).**Log into TenStreet (ATS).** • Update the candidate/employee profile. • Mark as terminated and archive file

## 2. Payroll & Records

Owner: HR Manager / Payroll Admin

- Open the Payroll Spreadsheet for the current pay period

• **Remove the terminated employee’s entry.**

- Open the Master Employee Spreadsheet

• **Remove the employee from the list.**

- **Confirm final paycheck details with CPS:**

• Final wages (regular hours, overtime, commissions if any).

• PTO payout, if applicable.

• Ensure correct timing of final pay (in compliance with AZ requirements).

## 3. Final Steps

Owner: HR Manager / Divisional Manager

- Exit Interview (if applicable)

• Schedule a short conversation with the employee.

• Document feedback and return to HR files.

- Internal Communication

• Notify the team of the employee’s departure (timing and tone based on situation).

• Ensure communication is respectful and professional.

- Personnel File

• **Archive the employee’s file in accordance with recordkeeping requirements (retain per state/federal law).**
