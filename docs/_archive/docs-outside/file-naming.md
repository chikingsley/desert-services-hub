# File Naming

## Dust Control Permits

Format: Dust_Control_Permit-[SiteName]-[PermitNumber]-[CompanyName]-[IssueDate].pdf

SiteName examples: Suncrest_Vista_At_Canyon_Trails, Modera_PV, TheReserve_AZ, etc.
PermitNumber: F052737 (Facility ID)
CompanyName: Layton_Construction_Company_LLC (or abbreviated if needed)
IssueDate: YYYY-MM-DD format (from Issue Date field)

Example: Dust_Control_Permit-Suncrest_Vista_At_Canyon_Trails-F052737-Weis_Builders_Inc-2025-07-17.pdf

## Dust Control Applications

Format: Dust_Control_Application-[SiteName]-[ApplicationNumber]-[CompanyName]-[CreatedDate].pdf

SiteName examples: Suncrest_Vista_At_Canyon_Trails, Modera_PV, The_Reserve_AZ, etc.
ApplicationNumber: D0059882 (from Dust Application Number)
CompanyName: Weis_Builders_Inc (or abbreviated if needed)
CreatedDate: YYYY-MM-DD format (from Created Date field)

Example: Dust_Control_Application-Suncrest_Vista_At_Canyon_Trails-D0059882-Weis_Builders_Inc-2025-07-14.pdf

## Contracts

Format: Contract-[SenderCompanyName]-[FullProjectName]-[Date].pdf

SenderCompanyName: The company sending the contract to Desert Services
FullProjectName: Complete project name (e.g., Suncrest_Vista_At_Canyon_Trails, Modera_Paradise_Valley)
Date: YYYY-MM-DD format

Example: Contract-Weis_Builders_Inc-Suncrest_Vista_At_Canyon_Trails-2025-12-10.pdf

## NOI Certificates (Notice of Intent - ADEQ Stormwater)

Format: NOI-[SiteName]-[LTFNumber]-[CompanyName]-[IssueDate].pdf

SiteName: Project/site name (e.g., PV_Redevelopment_Phase_5, Innovative_Commercial_Building)
LTFNumber: LTF# from the certificate (e.g., 110437, 113509)
CompanyName: Company the coverage is issued to
IssueDate: YYYY-MM-DD format (from Issued date on certificate)

Example: NOI-PV_Redevelopment_Phase_5-110437-BPR_Companies_LLC-2025-08-12.pdf

## NDC Certificates (No Discharge Certificate - ADEQ)

Format: NDC-[SiteName]-[CertificateNumber]-[CompanyName]-[IssueDate].pdf

SiteName: Project/site name
CertificateNumber: Certificate# from the document (e.g., 113801)
CompanyName: Company the certificate is issued to
IssueDate: YYYY-MM-DD format

Example: NDC-Kiwanis_Playground-113801-Caliente_Construction_Inc-2025-12-08.pdf

## Quotes

Format: Quote-[ClientCompany]-[ProjectName]-[Date].pdf

ClientCompany: Company the quote is for
ProjectName: Project name
Date: YYYY-MM-DD format

Example: Quote-Baker_Construction-Arterra-2025-12-17.pdf

## Invoices (Maricopa County Dust Application)

Format: Invoice-[ProjectName]-[InvoiceNumber]-[Date].pdf

ProjectName: Project name from the invoice
InvoiceNumber: Invoice number (e.g., IV086199)
Date: YYYY-MM-DD format (from Created Date)

Example: Invoice-PV_Block_C3-IV086199-2025-12-04.pdf

## General Rules

Use underscores or hyphens consistently (hyphens look cleaner).
No spaces, no special characters except - and _.
