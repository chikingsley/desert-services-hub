# Desert Services Sign Production Analysis

## Outsourcing vs. In-House UV Flatbed Production

* *Prepared:** December 2024  
* *Author:** Chi (Project Coordinator)  
* *Status:** Reference Document

- --

## Executive Summary

Desert Services currently outsources all compliance signage to Sandstorm Signs at an annual cost of **$123,750**. This analysis evaluates bringing sign production in-house using UV flatbed printing technology.

* *Key Finding:**At current volumes (850 signs/year), in-house production delivers**$70,000-$85,000 annual savings**after accounting for equipment, materials, labor, and maintenance. Equipment payback occurs in**10-14 months**.

| Scenario | Annual Cost | vs. Outsourcing |
|----------|-------------|-----------------|
| **Outsourcing (Current)** | $123,750 | — |
| **In-House (Part-Time Labor)**| $38,500 |**Save $85,250/year** |
| **In-House (Full-Time Dedicated)**| $53,700 |**Save $70,050/year** |

#### Alternative: Print Red on White (RECOMMENDED)

Instead of printing white text on red coroplast:

- Print RED graphics/text on WHITE coroplast
- Add red border/background as printed element
- Uses standard CMYK ink (no white ink premium)
- __Saves 75%+ on ink costs for fire access signs__
