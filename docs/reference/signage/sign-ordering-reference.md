| Fire Dept Access | 48" x 48" | Per contract or local requirements |
| Narrative/NOI Posting | - | Per project requirements |

- --

## Process

1. Collect required info per sign type (see below)
2. Email order to Sandstorm Signs (use template)
3. Log order in Sign Log
4. Coordinate delivery with install date
5. Install sign on site
6. Bill customer for signs

IMPORTANT: Get email confirmation before making sign changes (billing protection)

- --

## When to Update Signs

1. **Superintendent changes** → Update dust signs and fire access signs
2. **Responsible party changes** → Update all relevant signage
3. **Project on hold for months** → Check if anything changed (permits may need updating)
4. **Violation report received** → Review report and update any items that have changed
5. **Email confirmation received** → Any changes communicated via email

## Info to Request Before Emailing

- **Narrative board:** NOI (AZC #)
- **SWPPP sign/stickers:** NOI (AZC #)
- **Dust sign (any county):** County dust permit, contact name, contact phone number
- **Fire department access sign:** Site address, contact name, contact phone number
- **Delivery timing:** Pickup vs. delivery preference, need-by/install date

## Send Info

- **To:** [Kelli – Sandstorm Signs](mailto:kelli@sandstormsign.com)
- **Subject:** `Sign Order – {Qty}x {Sign Type}, {Qty}x {Sign Type}, ...`
