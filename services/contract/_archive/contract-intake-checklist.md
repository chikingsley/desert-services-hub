# Contract Intake Checklist (Human + Agent)

**Purpose:** Keep intake, reconciliation, and communications consistent when processing high volumes of contracts.

- --

## Minimum Required to Move Forward

- [ ] Contract PDF saved to SharePoint
- [ ] Estimate located in Monday (or explicitly flagged missing)
- [ ] Notion project created or linked and Intake Status set
- [ ] Contract value, retention, billing platform, and certified payroll status captured
- [ ] Reconcile outcome selected (Match / Revised Estimate / Clarification Needed)
- [ ] Internal contracts email draft ready
- [ ] All extracted facts include page/section citations

- --

## Human Tasks (Owner) - Required

- [ ] Save contract email and attachments to SharePoint
- [ ] Create/open Notion project and set Intake Status
- [ ] Attach contract PDF and estimate PDF in Notion
- [ ] Link Monday estimate and SharePoint folder
- [ ] Verify key terms (value, retention, billing platform, SOV, certified payroll)
- [ ] Review agent reconciliation summary for accuracy
- [ ] Update Reconcile Status and Handoff Status in Notion

## Human Tasks (Owner) - Nice to Have

- [ ] Add intake notes (what was searched, what was missing)
- [ ] Attach supporting docs (SOV, plans, specs) if provided
- [ ] Log red flags and scope questions in Notion
- [ ] If revised estimate required: create new estimate version and export PDF
- [ ] Attach revised estimate PDF to Monday and Notion

- --

## Agent Outputs - Required

- [ ] Extract contract value, dates, retention, billing platform, certified payroll, and SOV status with citations
- [ ] Extract key contacts (PM, superintendent, billing) with citations
- [ ] Compare totals and list removed vs added line items with citations
- [ ] Provide reconciliation summary and recommended outcome
- [ ] Flag red flags (scope ambiguity, missing SOV, certified payroll, unusual retention)
- [ ] Draft internal contracts email

## Agent Outputs - Nice to Have

- [ ] Extract site access instructions, site hours, and safety requirements with citations
- [ ] Provide a short list of unanswered questions
- [ ] Draft client clarification email (if needed)
